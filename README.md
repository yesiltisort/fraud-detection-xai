# Explainable AI for Credit Card Fraud Detection

## Overview

This repository contains my Master's thesis project completed as part of the MSc Data Science and Society at Tilburg University.

The project investigates how Explainable Artificial Intelligence (XAI) techniques can improve the interpretability of fraud detection models.

## Objectives

- Detect fraudulent transactions using machine learning
- Compare interpretable and black-box models
- Evaluate different XAI techniques

## Machine Learning Models

- Logistic Regression
- XGBoost

## Explainability Methods

- SHAP
- LIME
- Counterfactual Explanations

## Dataset

IEEE-CIS Fraud Detection Dataset (available on Kaggle)

The dataset is not included in this repository due to its size.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- SHAP
- LIME
